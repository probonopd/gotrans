# Gotrans

Gotrans is a golang command line tool, which is used to translate words or
sentences.

This tool base on **baidu translate api**.

## Installation

`go get github.com/skitta/gotrans`

## Usage

`gotrans hello`

`gotrans 'hello, world'`

`gotrans -t jp 'hello, world'`

more detial use `gotrans -h`
